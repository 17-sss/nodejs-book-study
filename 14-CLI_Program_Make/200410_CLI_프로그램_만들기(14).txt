* CLI 프로그램 만들기 (14)
    npm, nodemon이나 express-generator와 같이 커맨드라인 인터페이스 기반으로 동작하는 노드
    프로그램을 만들 것임.
    
    + 메모
        01. CLI (Command Line Interface)
            1) CLI는 콘솔 창을 통해서 프로그램을 수행하는 환경을 뜻함.
                반대 개념으로는 그래픽 사용자 인터페이스 GUI(Graphic User Interface) 가 있음.
            1-1) 리눅스의 셸이나 브라우저의 콘솔, 명령프롬프트등이 대표적인 CLI 방식 소프트웨어
            1-2) 원도, 맥 운영체제, 웹 애플리케이션 등이 대표적인 GUI 방식 소프트웨어
