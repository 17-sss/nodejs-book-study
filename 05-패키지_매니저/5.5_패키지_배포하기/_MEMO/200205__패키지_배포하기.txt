
* 패키지 배포
    1) cmd: cd [폴더경로]
    2) npm adduser
    3) npm publish
        - 패키지명 쓰지말기. 오류 나올 수 있음
    4) 예상대로 에러 검출 (패키지명: npmtest)
        - 이미 있기에 에러 검출
        - 누군가 이름을 사용하고 있는지 확인하려면 
            npm info [패키지명]
            + 404에러가 나온다면 사용해도 좋음

* 패키지 삭제
    - 주의사항: 패키지 배포 후, 24시간이 지나면 삭제 불가
    1) npm unpublish [패키지명] --force

* 패키지 배포 시 주의사항
    - 개인정보가 코드에 들어있는지 확인.
    - 이름을 선점하는 행위 삼가.
    - 기존에 있는 패키지와 비슷한 이름으로 새 패키지를 배포하거나 
        다른 패키지의 코드를 살짝 수정하여 새로 배포하는 경우
        원작자 허락을 받기.
