* 컬럼의 자료형 (일부 : 책)
    + INT : 정수를 의미
    + DOUBLE, FLOAT : 소수
    + CHAR(자릿수): 고정길이 문자형. 즉, CHAR(10)이면 반드시 길이가 10인 문자열만 넣어야함
        - CHAR에 주어진 길이보다 짧은 문자열을 넣는다면 부족한 지릿수만큼 스페이스가 채워짐
    + VARCHAR(자릿수): 가변길이 문자형. 즉, VARCHAR(10)일 경우 0~10길이의 문자열을 넣을 수 있음.
    + TEXT: 긴 글을 저장할 때 사용.
        - 몇백 자 이내의 문자열을 보통 VARCHAR 사용. 그보다 길면 TEXT
    + TINYINT: -127부터 128까지 정수를 저장할 때 사용.
        1 또는 0만 저장한다면 Boolean과 같은 역할 가능
    + DATETIME: 날짜와 시간에 대한 정보.
        - 날짜 정보만 담는 DATE와 시간정보만 담는 TIME 자료형도 있음.


* 자료형 뒤 옵션.
    + NULL & NOT NULL: 빈칸을 허용할지 여부를 묻는 옵션
    + AUTO_INCREMENT: 데이터가 늘어날때 마다 자동으로 숫자가 증가함. (회원가입이랄까)
    + UNSIGNED: 숫자 자료형에 적용되는 옵션. 
        - 숫자 자료형은 기본적으로 음수 범위를 지원하지만, UNSIGNED가 적용되면
            음수는 무시하고 INT의 경우 0~4294967295까지 저장 가능해짐.
    + ZEROFILL: 숫자의 자릿수가 고정되어 있을 때 사용.
        - INT(4)인데 숫자 1을 넣었다면 0001이 되는 식.
            ▷ 옵션명대로 0을 채우는 듯.
    + now() & CURRNET_TIMESTAMP: 현재 시각을 넣으라는 뜻.
    + PRIMARY KEY(컬럼명): 기본 키 설정. 
        - 기본 키: 로우를 대표하는 고유한 값을 의미 (주민등록번호와 비슷한 개념)
