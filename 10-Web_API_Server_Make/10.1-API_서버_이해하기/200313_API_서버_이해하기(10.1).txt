* API 서버 이해하기
    + API란?
        1. API의 개념
            - Application Programming Interface의 약어 
            - 다른 애플리케이션에서 현재 프로그램의 기능을 사용할 수 있게 허용하는 접점을 의미
        2. 웹 API
            - 웹 API는 다른 웹 서비스의 기능을 사용하거나 자원을 가져올 수 있는 창구
            - API를 열었다 or 만들었다는 표현은 다른 프로그램에서 현재 기능을 사용할 수 있게 허용하였음을 뜻함
            - 정보를 제공하고 싶은 부분만 API를 열어놓고, 제공하고 싶지 않은 부분은 API를 만들지 않는 것
                + 모든 사람이 정보를 가져갈 수 있는게 아니라, 인증된 사람만 일정 횟수 내에서 가져가게 제한 둘 수 있음
    + 크롤링
        1. 크롤링이란?
            - 웹사이트가 자체적으로 제공하는 API가 없거나, API 이용에 제한이 있을 때 사용하는 방법
            - 표면적으로 보이는 웹 사이트의 정보를 일정 주기로 수집해 자체적으로 가공하는 기술

        2. 주의사항 : 크롤링
            - 웹사이트에서 직접 제공하는 API가 아니므로 원하는 정보를 얻지 못할 가능성이 있음
            - 웹사이트에서 제공하기 원치 않는 정보를 수집한다면 법적인 문제가 발생할 수 있음
